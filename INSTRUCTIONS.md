# 🚀 Ron Translator — How To Open

## ✅ Fast Windows Method

1. Download the ZIP file.
2. Right-click the ZIP and click **Extract All**.
3. Open the extracted folder.
4. Double-click:

```txt
OPEN_RON_TRANSLATOR.bat
```

5. The Ron Translator will open in your browser automatically.

## ⚠️ Important

Do **not** run the BAT file from inside the ZIP preview. Extract the ZIP first, then run it.

## 📁 Main File

The main website file is:

```txt
index.html
```

You can also open `index.html` directly, but the BAT file is easier.
