# Ron Translator

A standalone, offline-friendly web translator for converting normal English text into **Ron runes** and translating Ron runes back into readable English.

## What This Project Does

- Converts English text to Ron runes
- Converts Ron runes back to English
- Works directly in a browser
- Can be installed as a PWA from supported browsers
- Supports copying and downloading translated text
- Includes local translation history
- Works without Python or Flask
- Includes immediate typing sounds, rune reveal sounds, a completion chime, and optional generated ambient background music

## How To Use

### Use the Website

1. Open the live website above, or open `index.html` in a browser.
2. Read the 4-second audio notice, then choose **Enter with Music** or **Enter Without Music**.
3. Type or paste text into the translator.
4. Pick the translation mode.
5. Copy or download the result.

### Put It on GitHub Pages

1. Create a GitHub repository, for example: `ron-translator`.
2. Upload all files from this project to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/(root)`.
6. Save the settings.
7. Your website will be published at a link similar to:

   `https://mrxavier53.github.io/ron-translator/`

### Install on Android

After the project is hosted online:

1. Open the website in Google Chrome on Android.
2. Tap the Chrome three-dot menu.
3. Tap **Install app** or **Add to Home screen**.
4. Open Ron Translator from your phone like a normal app.

## Current Animation and Audio Settings

- Translation begins about 20 milliseconds after input changes.
- Each translated character/rune appears every 20 milliseconds.
- A 4-second audio notice appears before the loading screen.
- Visitors can enter with optional low-volume generated ambient music or enter silently.
- Background music is created in the browser, so this project does not include a downloaded music track.
