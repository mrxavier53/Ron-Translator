## ✨ What is this?
A standalone Ron rune translator with a new original rune-chamber UI, floating Ron characters, ambient music, sound effects, offline support, history, file import, copy, and download.

## 🚀 GitHub Pages Setup
1. Upload the full folder to GitHub.
2. Enable **Settings → Pages**.
3. Select the branch and root folder.
4. Open the Pages URL.
5. Click **Enter with Music** or **Enter Without Music** and start translating.

## 🛠 Editing Guide
- Edit `index.html` for page text and layout.
- Edit files inside `assets/css/` for colors and styling.
- Edit files inside `assets/js/` for app logic and interactions.
- Keep paths relative so GitHub Pages works properly.
